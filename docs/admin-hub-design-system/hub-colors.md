# Hub Colors (Foundations)

**Last Updated:** 2026-05-21
**Source(s):** Atiom — Foundations (Figma); mirrored from `juliarainbowx/atiom-design-system/colors.md` (local clone at `~/design-system/`, last reconciled against commit `4c39a51` 2026-05-20). Content verified unchanged since 2026-04-28 snapshot — `status/new` rose token and full palette intact.

## Current Status

Color foundation for the **admin hub design system only**. Token format: `color/<palette>/<step>`. Steps go from 1000 (darkest) → 100 (lightest). Dark-themed token set (teal/darkgray) optimised for in-product UI. **Do not cross-wire with [[meridian]]** — Meridian uses indigo/copper/cream for external brand expression. Scope: admin hub.
