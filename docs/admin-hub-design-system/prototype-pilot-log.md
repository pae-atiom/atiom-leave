## How to use this log

Two columns of truth, both equally important:

- **Helped:** the gate caught something that would have caused a problem (rough HTML almost shipped to engineering, missing PRD link, design-system drift)
- **Friction:** the gate cost time without payoff, blocked legitimate work, or felt heavy

Quick entries. Two lines a day max per person. Timestamp + initials + one sentence. Don't write paragraphs — capture and move on. Synthesis happens at review.

Format:
```
- HH:MM / WM | helped: caught X
- HH:MM / YK | friction: had to override because Y
```
